# react-native-tiptap

A full fledged React Native Rich Text editor based on tiptap

## Installation

```sh
npm install react-native-tiptap
```

## Usage

```js
import Tiptap from "react-native-tiptap";

// ...

const result = await Tiptap.multiply(3, 7);
```
